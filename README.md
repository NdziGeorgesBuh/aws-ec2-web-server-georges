# AWS EC2 Web Server Project

## Overview
This project demonstrates how to launch an AWS EC2 instance and host a web server using Apache.

## What I did
- Created an EC2 instance (Amazon Linux)
- Configured Security Groups (opened ports 22 and 80)
- Connected via SSH using PowerShell
- Installed Apache web server
- Started and enabled Apache service
- Verified web server using public IP

## Technologies Used
- AWS EC2
- Linux (Amazon Linux 2023)
- Apache (httpd)
- SSH
- Networking (Security Groups, Ports)

## Result
Successfully deployed a working web server accessible via public IP.
